# AI-Driven Personalized Learning Path Generation for Engineering Students Through Skill Gap Analysis

This is the front-end user interface for the AI-Driven Personalized Learning Path Generation platform.

## Features Included
- AI Assessment & Skill Gap Analysis
- Interactive Learning Roadmaps & Modules
- Student & Admin Dashboards
- AI Mentor Integration
- Profile, Progress Tracking, Achievements, and Settings

## Running the code

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.
3. Run `npm run build` to build the production assets.
